# exo-blog-003

## Installation

- [AltoRouter](https://altorouter.com/)

Dans un fichier `composer.json`

```bash
{
    "require": {
        "altorouter/altorouter": "1.1.0"
    }
}
```
puis

```bash
composer install
```
